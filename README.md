#  Speech-to-Text Conversion

## Overview
A Python-based Speech-to-Text (STT) application built with Streamlit that transcribes LibriSpeech audio files using Google Speech Recognition. It evaluates transcription accuracy using Word Error Rate (WER), stores test history, and generates downloadable CSV reports.

## Technologies
- Python
- Streamlit
- SpeechRecognition
- Pandas
- JiWER

## Features
- Speech-to-text transcription
- WER accuracy calculation
- CSV report download
- Test history
